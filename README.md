# Multiprotocol TX Module Arduino IDE Boards
<img align="right" width=300 src="docs/images/multi.png" />

This repository contains the source for the Arduino IDE board definitions for the Multiprotocol TX Module.  

The main repository for the Multiprotocol TX Module firmware, including all documentation, is [here](https://github.com/pascallanger/DIY-Multiprotocol-TX-Module/).

## Installing the Multi 4-in-1 boards
The board definitions are installed using the Arduino IDE Boards Manager.

1. Open the Arduino IDE

2. Go to **File -> Preferences**, or press **Ctrl+Comma**

3. Locate the **Aditional Boards Manager URLs** field and paste in this URL: 

`https://raw.githubusercontent.com/pascallanger/DIY-Multiprotocol-TX-Module-Boards/master/package_multi_4in1_board_index.json`

**Note:** Multiple URLs are comma-separated.

For detailed instructions, see the installation documentation [here](https://github.com/pascallanger/DIY-Multiprotocol-TX-Module/blob/master/docs/Arduino_IDE_Boards.md)
